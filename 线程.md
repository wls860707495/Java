# 线程
## 线程的两种创建方式

      方法一：在Thread类继承
      ```java
      public class Test {
		    public static void main(String[] args) {
			    Runner1 r = new Runner1();
			    r.start();
			    for(int i=0;i<100;i++){
				    System.out.println("Main: "+i);
			    }
	    	}
	    }
      public class Runner1 extends Thread {
	      public void run(){
		      for(int i = 0;i<100;i++){
		    	System.out.println("Runner: "+ i);
		      }
      	}
      }
      ```java
